# mtm6201-final
Final Assignment - Calm Connect

## Project Overview
Calm Connect is a three-page responsive website built using Bootstrap and custom CSS. The goal of this project was to take a wellness app idea and turn it into a working website that shows features like mood tracking, journaling, community support, and helpful resources.

## Pages
- Home
- Community
- Resources

# Process
I started by looking at my design idea and figuring out how to turn it into a website layout. I used Bootstrap to build the structure and layout, then used my own CSS to adjust colours, spacing, and styling so it matched the calm and simple look I was going for.

# Challenges
One challenge was turning app-style screens into a website layout that still looked clean on desktop. I had to adjust how things were spaced and organized so it didn’t feel cramped.

Another challenge was keeping the design consistent across all pages. I fixed this by reusing the same styles for cards, buttons, and sections.

# What I Learned
I learned how to take a design idea and actually build it into a working website. I also got more comfortable using Bootstrap and customizing it with CSS. This project helped me understand responsive design better and how to structure pages properly.

# Accessibility
- Used semantic HTML (header, nav, main, footer)
- Added a skip link for accessibility
- Included alt text for images
- Made sure layout works on different screen sizes
